# 11047번: 동전 0
- [백준 그리디 알고리즘 11047번: 동전 0](https://www.acmicpc.net/problem/11047)

## 문제
- 동전은 총 N가지, 각각의 동전을 매우 많이 가지고 있음
- 동전을 적적히 사용해서 그 가치의 합이 K가 되도록 만들때 필요한 동전 개수의 최소값을 구하는 문제

## 입력
- 첫째 줄: N K
  - 1 <= N <= 10;
  - 1 <= K <= 100,000,000 (10^8, 1억) -> int

## 출력
- 첫째 줄: K원을 만드는데 필요한 동전 개수

## 예제
- 입력
```text
10 4200
1
5
10
50
100
500
1000
5000
10000
50000

10 4790
1
5
10
50
100
500
1000
5000
10000
50000
```
- 출력
```text
6

12
```
