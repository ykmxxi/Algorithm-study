# 16235번: 나무 재테크
- [백준 그래프 탐색 알고리즘 16235번: 나무 재테크](https://www.acmicpc.net/problem/16235)

## 문제
- N x N 크기의 땅, 1 x 1 크기의 칸으로 나누어 놓았음
  - 각각의 칸은 (r, c): r(행), c(열) r과 c는 1부터 시작
  - 격자형 배열 -> 그래프 문제
- 가장 처음에 양분은 모든 칸에 5만큼 들어있음
- M개의 나무를 구매해 칸에 심음. 같은 1 x 1 크기의 칸에 여러 개의 나무가 심어져 있을 수 있음
- 봄에는 나무가 자신의 나이만큼 양분을 먹고 나이가 1 증가.
  - 자신이 있는 칸의 양분만 먹을 수 있음 
  - 하나의 칸에 여러 개의 나무가 있으면 나이가 어린 나무부터 양분을 먹음
  - 땅에 양분이 부족해 자신의 나이만큼 먹을 수 없는 나무는 양분을 먹지 못하고 즉시 죽음
- 여름에는 봄에 죽은 나무가 양분으로 변하게 됨. 각각의 죽은 나무마다 나이를 2로 나눈 값이 나무가 있던 칸에 양분으로 추가 (소수점 아래는 버림)
- 가을에는 나무가 번식: 번식하는 나무는 나이가 5의 배수이어야 하며, 인접한 8개 칸에 나이가 1인 나무가 생김
  - 칸을 벗어난 경우 생기지 않음
- 겨울에는 로봇이 돌아다니며 땅에 양분을 추가한다. 각 칸에 추가되는 양분의 양은 A[r][c]이고 입력으로 주어짐
- K년이 지난 후 상도의 땅에 살아있는 나무의 개수를 구하는 문제

## 입력
- 첫째 줄: N M K
  - 1 <= N <= 10
  - 1 <= M <= N^2
  - 1 <= K <= 1,000
- 둘째 줄부터 N개의 줄에 A 배열의 값이 주어짐
  - 1 <= A[r][c] <= 100
- 다음 M개의 줄에는 상도가 심은 나무의 정보를 나타내는 세 정수 x y z가 주어짐
  - (x, y)위치에 z나이 나무가 심어져 있음

## 출력
- 첫째 줄: K년이 지난 후 살아남은 나무의 수
