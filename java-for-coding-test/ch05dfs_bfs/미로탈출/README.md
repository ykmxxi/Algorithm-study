# 미로 탈출

## 문제
- N x M 크기의 직사각형 형태의 미로
  - 동빈이의 처음 위치: (1, 1)
  - 미로의 출구 위치: (N, M)
  - 한 번에 한 칸씩 이동 가능
  - 괴물이 있는 부분: 0
  - 괴물이 없는 부분: 1
  - 미로는 반드시 탈출할 수 있는 형태
- 동빈이가 탈출하기 위해 움직여야 하는 최소 칸의 개수를 구하는 문제
  - 시작 칸과 마지막 칸을 모두 포함해서 계산

## 입력
- 첫째 줄: N M
  - 4 <= N, M <= 200
- 둘째 줄 부터 N개의 줄에 걸쳐 미로의 정보가 주어짐
  - 0 or 1

## 출력
- 첫째 줄: 최소 이동 개수

## 예제
- 입력
```text
// 예제 1
3 3
110
010
011

// 예제 2
5 6
101010
111111
000001
111111
111111
```
- 출력
```text
// 예제 1
5

// 예제 2
10
```
