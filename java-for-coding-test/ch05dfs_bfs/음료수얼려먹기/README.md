# 음료수 얼려 먹기
## 문제
- N x M 크기의 얼음 틀이 존재 (N행 M열)
  - 구멍이 뚫려 있는 부분: 0
  - 칸막이가 존재하는 부분: 1
- 구멍이 뚫려 있는 부분(0) 끼리 상하좌우로 붙어 있는 경우 서로 연결되어 있다고 가정
- 이때 생성되는 총 아이스크림의 개수를 구하는 문제
  - 0으로 되어 있는 부분을 연결시켜 그룹 수를 구하기

## 입력
- 첫째 줄: N, M
  - 1 <= N, M <= 1,000
- 둘째 줄 부터 N 개의 줄에 걸쳐 얼음틀의 형태가 주어짐
  - 0 or 1

## 출력
- 한 번에 만들 수 있는 아이스크림의 개수를 출력

## 예제
- 입력
```text
// 예제 1
4 5
00110
00011
11111
00000

// 예제 2
15 14
00000111100000
11111101111110
11011101101110
11011101100000
11011111111111
11011111111100
11000000011111
01111111111111
00000000011111
01111111111000
00011111111000
00000001111000
11111111110011
11100011111111
11100011111111
```
- 출력
```text
// 예제 1
3

// 예제 2
8
```
