# 정렬 실전 문제 2: 성적이 낮은 순서로 학생 출력하기

## 문제
- N명의 학생 정보가 존재할 때, 학생 정보는 학생의 이름과 성적으로 구분된다.
- 각 학생의 이름과 성적이 주어졌을 때, 성적이 낮은 순서대로 학생의 이름을 출력하는 문제
  - 성적이 동일하면 순서는 상관 없다.

## 입력
- 첫째 줄: 학생의 수
  - 1 <= N <= 100,000
- 둘째 줄부터 N개의 줄에 걸쳐 학생 정보가 입력된다.
  - 이름의 길이는 100 이하
  - 점수는 100 이하의 자연수

## 출력
- 첫째 줄: 성적이 낮은 순서대로 이름을 공백으로 구분해 출력한다.

## 예제
- 입력
```text
2
홍길동 95
이순신 77
```
- 출력
```text
이순신 홍길동
```
