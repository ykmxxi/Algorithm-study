# 이진 탐색 실전 문제 1: 부품 찾기

## 문제
- 동빈이네 매장에는 부품이 N개 존재
- 각 부품은 정수 형태의 고유한 번호가 존재
- 손님이 M개 종류의 부품을 대량으로 구매하겠다고 요청
- 이때 가게 안에 부품이 모두 있는지 확인하는 문제
- 예시
  - N = 5, [8, 3, 7, 9, 2] -> 동빈이네 매장에는 부품이 5개 존재, 8번, 3번, 7번, 9번, 2번
  - M = 3, [5, 7, 9] -> 손님이 3개 종류의 부품을 요청, 5번 7번 9번
  - 이때, 5번은 없고 7번, 9번이 존재 -> `no yes yes`

## 입력
- 첫째 줄: N
  - 1 <= N <= 1,000,000 (10^6)
- 둘째 줄: N개의 정수가 공백으로 구분해 주어짐
  - 1 <= 정수 <= 1,000,000
- 셋째 줄: M
  - 1 <= M <= 100,000
- 넷째 줄: M개의 정수가 공백으로 구분해 주어짐
  - 1 <= 정수 <= 1,000,000

## 출력
- 첫째 줄: 공백으로 구분해 각 부품이 존재하면 `yes`, 없으면 `no` 를 출력

## 예제
- 입력
```text
5
8 3 7 9 2
3
5 7 9
```
- 출력
```text
no yes yes
```
